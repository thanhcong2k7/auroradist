# Aurora Music Distribution Dashboard

A modern music distribution management platform built with React and TypeScript. Aurora provides artists and labels with a comprehensive dashboard to manage releases, track analytics, handle royalties, and engage with digital streaming platforms.

**Live Demo**: [auroradist.vercel.app](https://auroradist.vercel.app)

## Overview

Aurora Music Distribution (AuroraX) is a full-featured client dashboard designed for music distributors, artists, and labels. It offers intuitive tools for managing music releases, monitoring performance metrics, managing artist relationships, and tracking financial data.

## Features

### For Artists & Labels
- **Dashboard**: Real-time overview of releases, upcoming events, and key metrics
- **Discography Management**: Create, edit, and manage music releases with detailed metadata
- **Artist & Label Management**: Organize and manage artist profiles and label information
- **Track Management**: View and manage individual tracks across releases
- **Analytics**: Comprehensive analytics and performance metrics visualization
- **Wallet System**: Track royalties, payments, and financial transactions
- **Support**: Built-in support ticketing system
- **Settings**: Manage account preferences and profile information

### Admin Features
- **Admin Dashboard**: System overview and key metrics
- **Release Management**: Review and manage all releases across the platform
- **Revenue Analytics**: Detailed revenue and royalty tracking
- **DSP Management**: Digital Streaming Platform integration oversight
- **User Management**: Manage platform users and permissions
- **Label Management**: Oversee all labels on the platform
- **Support System**: Admin support ticket handling

## Tech Stack

### Frontend
- **Framework**: React 18.3.1
- **Language**: TypeScript 5.3.3
- **Routing**: React Router DOM 6.21.1
- **Styling**: Tailwind CSS 3.4.1
- **Build Tool**: Vite 5.0.10

### Libraries & Tools
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Charts**: Recharts 2.10.3
- **Icons**: Lucide React, React Icons
- **CSV Parsing**: PapaParse
- **Spreadsheet**: XLSX
- **Image Cropping**: React Easy Crop
- **UI Notifications**: Sonner
- **Styling Utils**: Autoprefixer, PostCSS
